# G-Scores Project

Đây là dự án **Laravel** dùng để thống kê và quản lý điểm thi THPT. Dưới đây là hướng dẫn cài đặt và chạy dự án.

---

## 1. Yêu cầu hệ thống
- **PHP** >= 8.0
- **Composer** >= 2.x
- **MySQL** hoặc **PostgreSQL** (tùy chọn)
- **Docker** (nếu bạn muốn chạy dự án bằng Docker)
